# Sales-Analysis-2014
A comprehensive data analysis project evaluating 2014 sales performance. Includes regional revenue trends, salesperson performance metrics and a dynamic Excel dashboard to support strategic business decision making.

Project Objective
To analyze the **2014 sales data** and provide a comprehensive report to aid management in making strategic business decisions for the coming year.

##  Key Results
* **Top Region:** North ($141,660 in revenue)
* **Top Salesperson:** Nancy Freehafer ($104,242)
* **Top Customer:** Company D ($67,180)
* **Peak Month:** December ($66,642)

##  Transaction Summary
The analysis shows that the business handles a high volume of smaller deals, with **218 transactions** falling within the **$0–$1,000**
